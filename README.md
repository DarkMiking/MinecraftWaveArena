# MinecraftWaveArena#
A school project, that uses a Discord Bot to summon a wave of mobs on a Minecraft Server. It is a fun way to train your combat skills in Minecraft.

## How to use the Arena

The usage of Wave Arena is very simple. When the Project was setup correctly on your PC you just need to know two commands. One of these commands is "!arena start", this command sets your world up for the incoming waves. You get teleported to the middle of the world where the arena is, your inventory gets cleared and you get the start loot. You can also use this command if you want to restart the Arena. The other command you need to use is "!wave start", this command starts the next wave. It only works when you have already used the first command and when there is currently no wave in progress. If you forget any of these commands you can also type !help and the bot will remind you. You might also find some additional quality of life commands in the !help command.
